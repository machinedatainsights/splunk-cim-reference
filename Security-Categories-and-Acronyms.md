## 🛡️ Endpoint & Network Security

| Acronym | Full Name                        | Description                                                                 |
|---------|----------------------------------|-----------------------------------------------------------------------------|
| **EDR** | Endpoint Detection and Response  | Detects and responds to threats on endpoints like servers or laptops.       |
| **XDR** | Extended Detection and Response  | Correlates data across endpoint, network, cloud for unified response.       |
| **NDR** | Network Detection and Response   | Detects anomalies and threats via network traffic analysis.                 |
| **NGAV**| Next-Gen Antivirus               | AI-based antivirus beyond signature matching.                               |
| **IDS** | Intrusion Detection System       | Detects malicious activity and policy violations.                           |
| **IPS** | Intrusion Prevention System      | Detects and actively blocks malicious traffic.                              |

## 👤 Identity & Access Management (IAM)

| Acronym | Full Name                        | Description                                                                 |
|---------|----------------------------------|-----------------------------------------------------------------------------|
| **IAM** | Identity and Access Management   | Manages user access and roles.                                              |
| **PAM** | Privileged Access Management     | Controls/administers elevated access rights.                                |
| **MFA** | Multi-Factor Authentication      | Requires multiple verification methods.                                     |
| **SSO** | Single Sign-On                   | One login for multiple systems/apps.                                        |
| **IdP** | Identity Provider                | Authenticates users for SSO/SAML/OIDC.                                      |

## 🔍 Threat Intelligence, Monitoring & Response

| Acronym   | Full Name                             | Description                                                                 |
|-----------|---------------------------------------|-----------------------------------------------------------------------------|
| **SIEM**  | Security Information and Event Mgmt   | Collects, correlates, analyzes logs for security insights.                 |
| **SOAR**  | Security Orchestration, Automation... | Automates security workflows and incident response.                        |
| **TIP**   | Threat Intelligence Platform          | Aggregates and enriches threat data from multiple sources.                 |
| **UEBA**  | User and Entity Behavior Analytics    | Detects anomalies based on user and entity behavior patterns.              |

## ☁️ Cloud & Application Security

| Acronym     | Full Name                             | Description                                                                 |
|-------------|---------------------------------------|-----------------------------------------------------------------------------|
| **CASB**    | Cloud Access Security Broker          | Enforces policies for cloud resource access.                                |
| **CSPM**    | Cloud Security Posture Management     | Identifies misconfigurations in cloud infrastructure.                       |
| **CWPP**    | Cloud Workload Protection Platform    | Protects VMs, containers, and serverless workloads.                         |
| **CNAPP**   | Cloud-Native App Protection Platform  | Combines CSPM and CWPP into one platform.                                   |
| **WAF**     | Web Application Firewall              | Blocks web-based attacks (e.g. XSS, SQLi).                                  |
| **DAST**    | Dynamic Application Security Testing  | Tests apps while running for vulnerabilities.                               |
| **SAST**    | Static Application Security Testing   | Analyzes source code for flaws.                                             |
| **RASP**    | Runtime Application Self-Protection   | Monitors and protects apps during runtime.                                  |

## 🧱 Infrastructure, Data & Asset Protection

| Acronym   | Full Name                             | Description                                                                 |
|-----------|---------------------------------------|-----------------------------------------------------------------------------|
| **VMS**   | Vulnerability Management System       | Scans and tracks software/hardware vulnerabilities.                        |
| **NAC**   | Network Access Control                | Controls device access to network resources.                                |
| **DLP**   | Data Loss Prevention                  | Prevents unauthorized data leakage.                                         |
| **EASM**  | External Attack Surface Management    | Identifies exposed digital assets for remediation.                          |

## 📊 Governance & Risk

| Acronym | Full Name                        | Description                                                                 |
|---------|----------------------------------|-----------------------------------------------------------------------------|
| **GRC** | Governance, Risk & Compliance    | Framework for managing IT risk and regulatory alignment.                    |
| **DRM** | Digital Rights Management        | Controls usage and access to digital content.                               |
| **PKI** | Public Key Infrastructure        | Enables encryption, signatures, certificates.                               |

## 🎯 Other Security Functions

| Acronym | Full Name                        | Description                                                                 |
|---------|----------------------------------|-----------------------------------------------------------------------------|
| **BAS** | Breach and Attack Simulation     | Emulates attacks to test detection/response capabilities.                   |
| **ZTA** | Zero Trust Architecture          | Requires continuous verification for all users and devices.                 |
